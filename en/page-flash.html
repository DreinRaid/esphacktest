ESP-HACK

Installation / flashing

Universal method

Open a web flasher in your browser, for example esp.huhn.me, connect the ESP over USB, and pick the firmware file from Releases. The 0x0 address is usually left as-is. Click PROGRAM; if needed, run ERASE first.

Automatic method (qESPHACK)

Download qESPHACK.exe from Releases or from the file list below. Run the utility and follow the steps in the window.

Using qESPHACK

Go to Releases and download qESPHACK.exe.

Then select your display and wait until installation (flashing) finishes.

qESPHACK finds the latest .bin for the selected display and downloads it automatically.
