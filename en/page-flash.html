ESP-HACK

Installation / flashing

Flash in the browser (this site)

Use Google Chrome or Microsoft Edge, a USB cable, and this page over HTTPS (e.g. GitHub Pages). Plug in your ESP32, pick the button for your display, and follow the installer dialog.

Binaries are pulled from Releases during the site deploy. If download fails, check the workflow asset names or use the external flasher below.

External web flasher

For example, open esp.huhn.me, connect the ESP over USB, and pick the firmware from Releases. Leave 0x0 as usual. Click PROGRAM; run ERASE first if needed.

Automatic method (qESPHACK)

Download qESPHACK.exe from Releases or from the file list below. Run the utility and follow the steps in the window.

Using qESPHACK

Go to Releases and download qESPHACK.exe.

Then select your display and wait until installation (flashing) finishes.

qESPHACK finds the latest .bin for the selected display and downloads it automatically.
