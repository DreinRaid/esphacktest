Firmware features

A compact feature list by category. Exact availability depends on firmware version and connected modules (CC1101 / NRF24 / IR / SD, etc.).

📶 Wi‑Fi

  • 📡 Beacon Spam — broadcast many fake SSIDs (clutter the Wi‑Fi list).
  • 🕸️ Evil Portal — run an AP with a captive portal landing page.
  • 🗺️ Wardriving — scan/log nearby networks.
  • 📦 Packets — Wi‑Fi packet tools (analysis/sending, where available).

🟦 Bluetooth / BLE

  • 📣 BLE-Spam — BLE advertisement spam (profiles for iOS/Android/Windows).
  • 🧪 BadBLE — BLE attack/emulation scenarios (build-dependent).
  • 🖱️ Mouse — BLE HID mouse emulation (if supported).

📻 Sub‑GHz (CC1101)

  • 📥 Read — receive & decode supported protocols.
  • 📤 Send — transmit decoded packets.
  • 🎙️ RAW — record & replay raw RF (record/send).
  • 🔎 Analyzer — signal analysis helpers.
  • 🔁 Bruteforce — code bruteforce for selected protocols (Came/Nice/Ansonic/Holtek/Chamberlain, etc.).
  • 🚫 Jammer — jamming (often illegal; use only where permitted).

🌈 Infrared (IR)

  • 📥 Read — capture IR codes.
  • 📤 Send — transmit IR codes.
  • 📺 TV / 📽️ PJ / ❄️ AC — ready-made presets (e.g. power off).

🧩 GPIO / modules

  • 🔑 iButton — read/write/config (with the required hardware).
  • 📶 NRF24 — jammer/spectrum/config modes (build- and region-dependent).
  • 🧷 Serial — in development (if enabled in your build).

⚙️ Settings

  • 🎨 Display color — UI theme/accent (display-dependent).
  • ⏲️ Standby time — sleep/standby timeout.
  • ℹ️ About — build/version info.
